# 🛡️ SAAS Admin - Platform Governance Documentation

As a **SAAS Admin**, you have absolute authority over the entire MCQ platform ecosystem. Below is a comprehensive list of functionalities available to you for institutional management, student governance, and system safety.

---

## 🏛️ Institutional (College) Management
- **College Onboarding**: Create new college accounts with unique codes and administrator credentials.
- **Feature Control**: Enable or disable specific platform features (e.g., Practice Mode, Proctoring) per institution.
- **Deep Analytics**: Access detailed statistics for any college, including total questions, exams conducted, and student performance.
- **Data Export**: Download institutional data (Students, Questions, Results) in Excel format.
- **Institutional Access**: View and manage all questions, exams, and results belonging to a specific college.

---

## 👥 Global User Governance
- **Centralized User List**: Monitor every user (Student, Teacher, Admin) across all institutions from a single dashboard.
- **Identity Management**: Edit user profiles (Name, Email, Role) globally.
- **Operational Toggles**: Activate or deactivate any user account instantly to control platform access.
- **Root Password Reset**: Securely reset the password for any user on the platform.
- **Soft Deletion**: Move problematic or inactive users to the "Recovery Panel" instead of deleting them immediately.

---

## 📝 Exam & Question Oversight
- **Global Question Bank**: View and manage every question uploaded to the platform.
- **Exam Monitoring**: Track all live and past exam sessions.
- **Result Visibility**: Access student attempts and scores for any exam.
- **Governance Controls**: Toggle exam availability and result visibility for students.
- **Purge Actions**: Soft-delete outdated questions or exams to keep the platform clean.

---

## 🔄 Recovery & Permanent Purge
- **Recovery Engine (Trash)**: A central "Graveyard" where all deleted Colleges, Users, Exams, Questions, and Results are stored.
- **Restoration**: One-click recovery for any soft-deleted item, bringing it back to live status.
- **Permanent Deletion**: The ultimate power to purge records from the database entirely (ONLY available for items already in the Trash).

---

## 🕵️ Accountability & Audit Trail
- **Action Tracking**: Every administrative action (Create, Update, Delete, Restore, Password Reset) is automatically logged.
- **Audit Logs Dashboard**: Search and filter the platform history to see **Who did What, When, and to Whom**.
- **Transparency**: Identify the source of any data change, ensuring full accountability across all institutions.

---

## 📊 Platform Analytics
- **Global Overview**: Real-time stats on total institutions, active users, questions in the bank, and exams completed.
- **Growth Tracking**: Monitor platform usage trends and institutional activity.

---
> [!IMPORTANT]
> Some actions, like **Permanent Deletion**, are destructive and cannot be undone. Always verify the item in the Recovery Panel before purging it forever.
